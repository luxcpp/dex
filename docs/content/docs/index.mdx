# Lux DEX

High-performance C++17 order book and matching engine for the Lux Network DEX.

## Overview

Lux DEX provides a blazing-fast Central Limit Order Book (CLOB) implementation optimized for high-frequency trading workloads. Key features include:

- **Sub-microsecond latency** - Optimized for HFT workloads
- **Price-time priority** - FIFO matching with configurable tie-breaking
- **Lock-free data structures** - Minimal contention under high load
- **Multi-asset support** - Handle thousands of trading pairs
- **C/Go bindings** - Native integrations for other languages

## Quick Start

```cpp
#include <lux/orderbook.hpp>
#include <lux/engine.hpp>

lux::Engine engine;
engine.add_order({
    .side = lux::Side::Buy,
    .price = 50000.00,
    .quantity = 1.5,
    .symbol = "BTC-USD"
});
```

## Building

```bash
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)
```
